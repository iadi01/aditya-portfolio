# Aditya Portfolio | Aditya Sharma Portfolio

Welcome to the official repository for **[Aditya Portfolio](https://aadi-sharma.dev/)** (also known as the **[WTF Aadi Portfolio](https://aadi-sharma.dev/)**) created by **Aditya Sharma**, a Full-Stack Web Developer and BCA Student from Jamshedpur, India.

This developer website showcases interactive features, clean typography, and optimized web layouts.

**Live Site:** 👉 **[https://aadi-sharma.dev/](https://aadi-sharma.dev/)**

---

## 🚀 Key Features

*   **Neobrutalist Styling:** Bold borders, primary colors, and flat offset shadows built on Tailwind CSS v4.
*   **Custom Interactive Terminal:** Retro command-line interface supporting custom commands (`whoami`, `skills`, `projects`, `contact`).
*   **Dynamic Sound Synthesis:** Real-time synthesizer audio feedback on hover and navigation clicks.
*   **SEO & Core Web Vitals Optimized:** Zero Cumulative Layout Shift (CLS), structured JSON-LD schemas, and pre-rendered HTML fallbacks.
*   **Interactive 404 Console:** Custom 404 page featuring automated system diagnostics.

---

## 🛠️ Tech Stack

*   **Core:** React 19, JavaScript (ES6+), HTML5, CSS3
*   **Styling:** Tailwind CSS v4, Vanilla CSS
*   **Build Tool:** Vite 8
*   **Animations:** Framer Motion 12
*   **Icons:** React Icons
*   **Deployment:** Vercel CDN
